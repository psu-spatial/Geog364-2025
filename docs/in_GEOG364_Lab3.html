MAIN ANALYSIS

Goal: understand which factors influence house prices in Sindian District, New Taipei City, Taiwan, and communicate results to a non-technical reader.

Read and prepare the data

  1. Download and inspect the Lab03_house.xlsx dataset

  2. Read into R (read_excel) and check structure

  3. Clean and rename columns if needed

  4. Convert to spatial object (st_as_sf, crs = 4326)

Define the study and data context

  1. State the decision problem (friend/boss house-buying question)

  2. Define the population of interest and discuss representativeness

  3. Describe key variables and their units (response = House.Price). Inc Summary statistics and distribution of House.Price (mean, median, range, IQR – and histogram)

  4. Note any data limitations (temporal coverage, sample bias, missing variables)

  5. Discuss spatial fallacies/properties e.g. locational fallacy? (nope) etc etc

Add more data

Elevation raster (get_elev_raster)

  1. Get elevation

OpenStreetMap

  1. OpenStreetMap features: railway lines, stations, parks, roads

Spatial analysis and mapping

  1. Quick interactive map of points coloured by price (tmap, qtm)

  2. Main map from my example

  3. Describe spatial patterns

  4. Simple linear regression (price ~ latitude)
    Use maps to explain the scatterplots e.g. should I just walk north

Conclusions

  1. What processes are happening? Summarise main drivers of price from combined evidence Discuss practical implications for a house-buyer
